# Threat Intelligence Mini Reports

This repository contains short, structured threat intelligence briefs designed to help analysts, students, and security professionals quickly understand active threats, malware families, and adversary behaviors. Each report includes:

- MITRE ATT&CK mappings  
- Tactics, techniques, and procedures (TTPs)  
- Indicators of compromise (IOCs)  
- Detection opportunities  
- Risk impact analysis  
- Recommended mitigations  

Reports are organized by category and written using a consistent template for clarity and repeatability.
